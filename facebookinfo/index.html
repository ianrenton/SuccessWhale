Discontinuation of Facebook Support

What happened?

On April 30th 2015, Facebook started enforcing a manual review process whereby each application must be individually reviewed for each permission it requests. Some permissions that are vital for SuccessWhale, such as "read_stream", are no longer granted to third-party applications. You can read about the implications on my blog.

What does this mean for SuccessWhale?

Reading Facebook feeds in SuccessWhale will no longer work. Unfortunately there seems to be nothing we can do about this.

Is there a work-around?

There is, but you're not going to like it. Facebook automatically allows all permissions if the user requesting them is the owner of the application. This means that if you run your own copy of SuccessWhale — including the API and database — then Facebook will work as intended for you (and only you).
